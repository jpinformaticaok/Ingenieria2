## 📚 Descripción

Este repositorio contiene los trabajos prácticos, ejercicios y proyectos desarrollados en la materia **Ingeniería II** de la carrera **Tecnicatura Superior en Desarrollo de Software**.

A lo largo de la cursada se explorarán las tecnologías fundamentales del desarrollo web frontend, avanzando desde la estructura y el diseño estático hasta la interactividad y el uso de frameworks modernos.

## 📋 Contenidos del Programa

### 🔷 Módulo 1 — HTML5
- Estructura básica de un documento HTML
- Etiquetas semánticas (`header`, `nav`, `main`, `footer`, etc.)
- Formularios y validaciones nativas
- Accesibilidad web

### 🔷 Módulo 2 — CSS3
- Selectores, especificidad y cascada
- Box Model y posicionamiento
- Flexbox y CSS Grid
- Animaciones y transiciones
- Diseño responsivo con Media Queries

### 🔷 Módulo 3 — JavaScript
- Fundamentos del lenguaje (variables, funciones, estructuras de control)
- Manipulación del DOM
- Eventos e interactividad
- Fetch API y consumo de datos externos
- ES6+ (arrow functions, destructuring, módulos)

### 🔷 Módulo 4 — Bootstrap
- Sistema de grillas
- Componentes (navbar, cards, modals, etc.)
- Utilidades y helpers
- Personalización con variables Sass

## 🚀 Cómo usar este repositorio

1. **Cloná el repositorio:**
   ```bash
   git clone https://github.com/tu-usuario/ingenieria-II.git
   cd ingenieria-II
   ```

2. **Navegá a la carpeta del trabajo que quieras ver:**
   ```bash
   cd trabajos-practicos/TP01-HTML-semantico
   ```

3. **Abrí el archivo `index.html` en tu navegador** o utilizá la extensión **Live Server** de VS Code para verlo en tiempo real.
